# TikTok Verified Status Prediction: Discovery-to-Action Pipeline

## Project Summary
This project addresses a growing content report backlog for TikTok's Operations Lead, Maika Abadi, by deploying a machine learning classification framework. By leveraging video-level metadata and user engagement metrics, we implement a Logistic Regression model to predict whether a reported video's creator is verified or unverified. This predictive capability allows the operations team to programmatically distinguish between systematic violations and benign user opinions, driving efficient operational resource allocation.
